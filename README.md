# 🏥 Simulador de Transações de Saúde

## 📌 O que o código contém

* **Classe Paciente**

  * Nome e idade
  * Validação de idade e nome

* **Classe Medico**

  * Nome e CRM
  * Validação de dados

* **Classe Medicamento**

  * Nome e dosagem
  * Retorno de descrição

* **Classe Consulta**

  * Contém Paciente, Medico e lista de Medicamentos
  * Método para adicionar medicamentos
  * Método `gerar_resumo()` com as informações da consulta

## 🔐 Conceitos aplicados

* Encapsulamento (atributos privados + validação)
* Composição (Consulta possui outros objetos)
